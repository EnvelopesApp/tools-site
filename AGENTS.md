# Instructions for Codex

This repository is the public Toolshelf website. CleanCut app updates do not
normally require a website edit because the Download button uses the stable
GitHub Releases URL.

Follow `video_censor_desktop/RELEASE.md` for app releases. Do not change the
download button to an unsigned build, legacy payload, or version-specific URL.
The owner has granted standing authorization to publish completed website
changes after validation; always verify the GitHub Pages build and live page.
